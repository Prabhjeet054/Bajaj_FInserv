## Running Locally

### Backend
cd backend
npm install
npm run dev    # runs on http://localhost:3000

### Frontend
cd frontend
npm install
npm run dev    # runs on http://localhost:5173

## API Usage

Content-Type: application/json

Body: { "data": ["A->B", "A->C", "B->D"] }
